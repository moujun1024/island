# 用户协议

## 1. 版权声明

### 1.1 转载内容
- 明确标注为"转载"的文章，版权归原作者所有
- 用户应确保已获得原作者的转载授权
- 因转载内容引发的版权纠纷由用户自行承担

### 1.2 原创内容
- 未标注为转载的内容视为用户原创
- 用户保留所有版权，但授予本应用展示权
- 禁止未经授权全文复制或商业性使用

### 1.3 合理使用
- 允许个人学习目的的片段引用
- 引用需注明出处并保持内容完整性

## 2. 用户责任
- 保证上传内容不侵犯第三方权益
- 对标注的转载/原创属性真实性负责

## 3. 违规处理
- 发现侵权内容将立即删除
- 多次违规将暂停账户权限
