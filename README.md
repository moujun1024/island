## 使用教程
### 1. 获取项目
```bash
git clone https://github.com/your-repo/island.git
cd island
```

### 2. 修改项目名称
- 全局搜索替换"Island"为您自己的项目名
- 更新以下文件中的标识信息：
  - application_data.json
  - 所有.md文件
  - 配置文件

### 3. 替换文章内容
- 删除/替换article目录中的原创文章
- 添加您自己的文章内容

### 4. 部署使用
- 将修改后的项目部署到您的服务器
- 或使用静态网站托管服务

## 注意事项
1. 必须更改项目名称
2. 必须替换原创文章内容
3. 文件结构可直接使用
